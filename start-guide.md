# 🚀 智慧采购系统 - 启动指南

## 📋 快速启动

智慧采购系统的AI增强功能已经完成，以下是您需要做的步骤：

### 第一步：启动服务

系统提供了演示服务，您可以通过以下方式访问：

```bash
# 访问系统演示页面
http://localhost:8080/demonstration/demo.html
```

### 第二步：体验AI功能

#### 🤖 AI聊天功能
打开演示页面，您可以与AI助手进行自然语言对话，测试以下功能：

- **问候对话**: "你好" 查看欢迎导览
- **产品咨询**: "我需要服务器" 获取选型建议
- **价格判断**: "这个价格合理吗？" 了解分析方法
- **合同分析**: "合同有哪些风险？" 识别条款问题
- **需求审查**: "需求完整性检查" 查看评估标准

#### 🎯 快速操作按钮
页面右侧提供了快捷操作：
- 「服务器选型」- 查看主流产品推荐
- 「价格参考」- 了解市场价格区间
- 「合同分析」- 学习风险识别要点
- 「需求检查」- 评估文档完整性

### 第三步：深入体验
