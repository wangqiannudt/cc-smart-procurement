# 采购流程指南

## 1. 需求确认阶段
在采购前，需要明确以下内容：
- 用途和应用场景
- 功能边界和功能模块
- 关键性能指标（KPI）
- 预算范围
- 交付时间要求

## 2. 供应商选择
选择供应商时应考虑：
- 供应商资质和信誉
- 产品或服务的技术参数
- 历史业绩和客户评价
- 售后服务能力
- 价格合理性

## 3. 价格谈判
价格谈判技巧：
- 收集市场价格参考
- 了解供应商成本结构
- 明确付款方式和账期
- 注意隐藏成本（运输、安装、培训等）

## 4. 合同审核要点
合同审核重点关注：
- 合同金额和付款方式
- 交付范围和交付期限
- 验收条款和验收标准
- 质保条款和售后服务
- 违约责任和争议解决
